# Leo Class 英語大會考重新規劃

## 方向

這個版本改成「離線網頁課程」，不依賴 Google 試算表。學生從入口選擇年級後，進入該年級的英語大會考練習。課程可以直接放在 GitHub Pages，也可以嵌入 Google 協作平台。

主軸是聽、說、讀、寫都要視覺化，讓學生每天可以短時間反覆念、反覆聽，最後把單字放進簡單故事句裡應用。

## 一年級調整

一年級不再只是英文字母筆畫，而是改成自然發音入門。

一年級關卡：

1. Letter Sounds：A-Z 字母音與代表單字。
2. Listening Flip：聽自然發音單字，翻牌配對英文與意思。
3. Word Exam：聽音後選出正確單字。
4. Speaking Practice：跟念字母音與單字。
5. Reading Sentences：讀簡短句。
6. Story Mission：用一年級單字組成簡單故事句。

## 二到六年級課程

二到六年級每個年級都採用六個任務：

1. Daily Word Cards：全部單字分成 Day 1 到 Day 7，每天看圖、聽音、跟念單字與短句。
2. Listening Flip：聽 TTS，翻牌找出英文單字卡與中文圖示卡。
3. Speaking Practice：聽一句、跟念一句。
4. Reading Sentences：讀例句，選出缺少的單字。
5. Writing Quest：依提示拼出單字。
6. Story Mission：把單字放進簡單故事句。

## 一週七天

所有正式單字都會出現在課程中，不抽掉、不只做範例。系統把每個年級的單字平均分成七天，每天念一部分。學生可以跟著今日建議日練習，也可以一次瀏覽完整七天。

## 關卡開放

所有關卡目前都直接開放，不需要完成前一關才能進下一關。這樣老師可以快速測試每個任務，也方便課堂上直接指定學生進入某一個活動。

## 圖像與聲音

- 單字卡使用圖示輔助記憶。
- 使用瀏覽器 Web Speech API 播放美式英文 TTS。
- ChatGPT 產生的圖片放在 `assets/`，作為地圖、角色與關卡視覺。
- 不需要雲端資料庫，學生資料只存在本機 `localStorage`。
