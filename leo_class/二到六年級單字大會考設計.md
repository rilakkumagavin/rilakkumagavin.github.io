# 二到六年級單字大會考設計

## 設計來源

青溪國小數位資源網英語專區列有：

- 單字大會考－二年級
- 單字大會考－三年級
- 單字大會考－四年級－1／2／3
- 單字大會考－五年級
- 單字大會考－六年級

目前已接入使用者提供的正式來源：

- 二年級：`單字大會考＿二年級(20230911).pdf`，擷取 26 句大會考句子與 53 個目標單字。
- 三到六年級：`英語會考600單 20221228.xlsx`，使用 `3`、`4`、`5`、`6` 工作表。

正式字表獨立放在 `word-data.js`，互動邏輯仍在 `app.js`。日後如果字表更新，只要替換 `word-data.js` 即可。

## 共用任務架構

二到六年級每個年級使用六個任務：

| 任務 | 功能 | 互動 |
|---|---|---|
| Daily Word Cards | 單字暖身與美式 TTS | 全部單字分成七天，每天點卡聽讀 |
| Listen & Pick | 聽力辨識 | 聽 TTS 後四選一 |
| Speaking Practice | 口說跟讀 | 聽一句，跟念三次，自我打卡 |
| Reading Sentences | 閱讀應用 | 讀句子，選出適合單字 |
| Writing Quest | 拼字練習 | 點字母拼出單字 |
| Story Mission | 故事應用 | 讀簡單故事句，點選故事單字 |

## 七天每日跟讀

每個年級的全部單字都會出現在 `Daily Word Cards`，系統平均切成 Day 1 到 Day 7。

每日建議流程：

1. 看圖 3 秒。
2. 按下單字卡聽美式英文 TTS。
3. 跟著念三次。
4. 說出中文意思。
5. 週末複習 Day 1 到 Day 7。

## 資料格式

`word-data.js` 中每個單字使用：

```js
{ "num": 1, "word": "aunt", "zh": "姑.姨.嬸.伯/舅母", "letter": "A" }
```

二年級因為來源是句子，所以多一個 `sentence`：

```js
{ "num": 1, "word": "cat", "zh": "貓", "letter": "C", "sentence": "A cat takes a nap by the nest." }
```

圖示由 `app.js` 的 `iconFor()` 依單字與中文意思自動分配，之後若有正式圖片，也可以把資料欄位改成圖片檔名，再擴充畫面顯示方式。
