# 二到六年級單字大會考設計

## 設計來源

青溪國小數位資源網英語專區列有：

- 單字大會考－二年級
- 單字大會考－三年級
- 單字大會考－四年級－1／2／3
- 單字大會考－五年級
- 單字大會考－六年級

目前已接入使用者提供的正式來源：

- 二年級：`單字大會考＿二年級(20230911).pdf`，擷取 26 句大會考句子與 53 個目標單字。
- 三到六年級：`英語會考600單 20221228.xlsx`，使用 `3`、`4`、`5`、`6` 工作表。

正式字表獨立放在 `word-data.js`，互動邏輯仍在 `app.js`。日後如果字表更新，只要替換 `word-data.js` 即可。

## 共用任務架構

二到六年級每個年級使用五個任務：

| 任務 | 功能 | 互動 |
|---|---|---|
| Word Bank | 單字暖身與美式 TTS | 點卡聽讀 |
| Listen & Pick | 聽力辨識 | 聽 TTS 後四選一 |
| Meaning Match | 中英語意連結 | 看中文或圖示選英文 |
| Spelling Quest | 拼字練習 | 點字母拼出單字 |
| Word Exam | 綜合大會考 | 混合聽力與語意測驗 |

## 資料格式

`word-data.js` 中每個單字使用：

```js
{ "num": 1, "word": "aunt", "zh": "姑.姨.嬸.伯/舅母", "letter": "A" }
```

二年級因為來源是句子，所以多一個 `sentence`：

```js
{ "num": 1, "word": "cat", "zh": "貓", "letter": "C", "sentence": "A cat takes a nap by the nest." }
```

圖示由 `app.js` 的 `iconFor()` 依單字與中文意思自動分配，之後若有正式圖片，也可以把資料欄位改成圖片檔名，再擴充畫面顯示方式。
