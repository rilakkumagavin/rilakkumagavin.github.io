# Leo Class 離線遊戲課程重新規劃

## 方向

把目前分散在各科資料夾與 GitHub Pages 輸出資料夾中的遊戲課程，整理成一個「離線可開啟、可嵌入協作平台、可上傳 GitHub Pages」的中型網頁系統。

系統不使用 Google 試算表作為資料來源。課程資料直接放在網頁內的 JavaScript 物件，教師可用文字編輯器修改，學生只要打開網頁即可使用。

## 入口架構

第一層是學生入口：

1. 選擇年級
2. 查看該年級課程島
3. 進入 RPG 關卡
4. 完成任務取得知識徽章
5. 產生學習成果卡

目前優先重做一年級英語，來源是青溪國小數位資源網「英語專區」中的「單字大會考－一年級」。截圖與頁面顯示一年級核心是 `Letter Recognition · US English TTS`，因此第一版改為「字母辨識、TTS 朗讀、聽音選字母、單字大會考」，不是 Google 試算表，也不是單純自然發音筆畫練習。

## 年級規劃

| 年級 | 主題定位 | 第一版狀態 |
|---|---|---|
| 一年級 | 單字大會考：Letter Recognition + US English TTS | 優先製作 |
| 二年級 | 字母、單字與生活句型橋接 | 預留入口 |
| 三年級 | 目前既有國語、數學、自然、社會、健體 RPG | 可逐步接回 |
| 四年級 | 閱讀理解與跨科探究 | 預留入口 |
| 五年級 | 任務式專題與多步驟推理 | 預留入口 |
| 六年級 | 畢業前整合挑戰與作品集 | 預留入口 |

## 一年級大會考定位

一年級不再以「寫出 A、B、C 的筆畫」為核心，而改為英語專區單字大會考型態：

- 看見字母：能辨認 A-Z。
- 聽見字母：能使用美式英文 TTS 聽字母名稱。
- 聽音選字母：能依朗讀選出正確字母。
- 看圖讀單字：能辨認一年級 A-Z 代表單字。
- 成果紀錄：能產生大會考成果卡。

第一版先做 3 個任務：

1. Letter Recognition：A-Z 字母按鈕與 US English TTS。
2. Listen & Pick：聽朗讀，選出正確字母。
3. Word Exam：看圖、聽 TTS，選出正確單字。

## 技術型態

- `index.html`：學生入口與課程畫面。
- `app.js`：年級資料、關卡資料、互動邏輯。
- `style.css`：版面與觸控介面。
- `assets/`：圖片素材，可使用 ChatGPT 產生後放入。
- `localStorage`：保存學生在同一台裝置上的進度。

如果要內嵌到協作平台，可以直接使用 `index.html`，或把整個資料夾上傳到 GitHub Pages 後用網址嵌入。

## 圖片策略

可以使用 ChatGPT 產生圖片，但素材表要固定，避免每次風格不同。

| 素材 | 建議檔名 | 用途 |
|---|---|---|
| 年級選擇大地圖 | `map_main.png` | 首頁背景 |
| 學生主角 | `hero_student.png` | 首頁與任務畫面 |
| 英語導師 NPC | `npc_teacher.png` | 任務提示 |
| 一年級任務背景 1 | `stage_phonics_a.png` | Letter Recognition |
| 一年級任務背景 2 | `stage_phonics_b.png` | Listen & Pick |
| 一年級任務背景 3 | `stage_phonics_c.png` | Word Exam |

生成圖時避免圖片內出現文字，文字交給網頁呈現。
