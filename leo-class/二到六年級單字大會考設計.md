# 二到六年級單字大會考設計

## 設計來源

青溪國小數位資源網英語專區列有：

- 單字大會考－二年級
- 單字大會考－三年級
- 單字大會考－四年級－1／2／3
- 單字大會考－五年級
- 單字大會考－六年級

目前公開頁面可確認頁面存在，但嵌入題庫內容未直接露出。因此第一版採「可替換正式字表」的離線架構，先放示範字庫，之後可把正式單字表貼進 `app.js` 的 `gradeVocab`。

## 共用任務架構

二到六年級每個年級使用五個任務：

| 任務 | 功能 | 互動 |
|---|---|---|
| Word Bank | 單字暖身與美式 TTS | 點卡聽讀 |
| Listen & Pick | 聽力辨識 | 聽 TTS 後四選一 |
| Meaning Match | 中英語意連結 | 看中文或圖示選英文 |
| Spelling Quest | 拼字練習 | 點字母拼出單字 |
| Word Exam | 綜合大會考 | 混合聽力與語意測驗 |

## 年級主題

| 年級 | 第一版主題 | 後續替換方式 |
|---|---|---|
| 二年級 | 教室與學用品 | 替換為正式二年級字表 |
| 三年級 | 家人、感受、地點 | 替換為正式三年級字表 |
| 四年級 | 食物、飲料、日常 | 可拆成 4-1、4-2、4-3 |
| 五年級 | 星期、時間、季節 | 替換為正式五年級字表 |
| 六年級 | 地點、旅行、總複習 | 替換為正式六年級字表 |

## 資料格式

每個單字使用：

```js
["word", "中文意思", "開頭字母", "圖示"]
```

例如：

```js
["book", "書", "B", "📘"]
```

正式字表若有圖片，也可以把圖示改成圖片檔名，再擴充畫面顯示方式。
