# 社會 RPG 圖片素材規劃

## 美術風格

- 風格：明亮溫暖的兒童繪本風，帶有桌遊地圖與教育遊戲感。
- 色彩：青綠、暖黃、珊瑚橘、草綠、淡紫作為分區色，不使用陰暗恐怖氣氛。
- 文字：圖片中避免生成中文字或英文字，所有文字由 HTML 顯示。
- 構圖：背景保留上方或中央 UI 空間，角色與互動物件不要遮住主要操作區。
- 生成方式：使用 ChatGPT 圖片生成，再放入 `社會/assets/`。

## 接入規格

- 所有專案使用圖片放入 `社會/assets/`。
- 透明角色素材先以 #00ff00 去背流程處理，再輸出 PNG。
- 背景素材用 CSS `background-image` 套到首頁、章節與任務頁。
- 若圖片載入失敗，介面仍保留原本 CSS 背景與文字內容。
